Workflow

🗄️

1. Database Configuration

Our cloud database stores all categories and search terms, replacing the manual Excel sheet approach.

What's happening:

  • • Categories and search terms are pre-configured
  • • Custom article links can be added during the week
  • • All data is stored securely in the cloud
📰

2. Click "Fetch All News" Button

One click starts the automated news gathering process for all categories.

Process initiated:

  • • System loops through each category
  • • Executes search terms automatically
  • • Begins multi-level article fetching
🌐

3. Advanced Web Scraping

Multi-layered approach to access articles from various websites with different restrictions.

Level 1: Direct

Standard HTTP requests

Level 2: Proxy

Bypass basic restrictions

Level 3: Chrome Extension

Human-like interaction

4. AI-Powered Summarization

GPT-4o deployed on Microsoft Azure processes each article with custom instructions.

AI Processing:

  • • Extracts clean text from HTML
  • • Applies custom system instructions
  • • Generates title and concise summary
  • • Maintains consistent formatting
🔧

5. Chrome Extension Action

One more click on the Chrome extension resolves any remaining restricted articles.

Extension capabilities:

  • • Simulates human browser behavior
  • • Bypasses advanced website restrictions
  • • Handles JavaScript-heavy sites
  • • Completes the article collection
✅

6. Article Selection Interface

All articles are displayed in an organized table where you can select which ones to include.

Selection features:

  • • View all fetched articles by category
  • • Preview summaries before selection
  • • Easy checkbox selection interface
  • • Category-wise organization

7. Generate Knowledge Note

Final click creates a summary of summaries for each category and generates the structured Word document.

Final processing:

  • • Combines selected article summaries by category
  • • Creates comprehensive category overviews
  • • Formats into professional Word document
  • • Ready for email distribution
